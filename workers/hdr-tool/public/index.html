Using Masks

What is a Mask?

A mask controls where the HDR effect is applied:

  • White areas = HDR effect applied
  • Black areas = Original image preserved
  • Gray areas = Partial effect (blend)

Built-in Tools

Use the Brush (B) to paint areas for HDR effect. Use the Eraser (E) to remove from the mask. These are best for simple selections.

Importing Professional Masks

For complex masks (hair, edges, detailed selections), create them in professional tools:

  1. Photoshop: Use Select > Subject or Quick Selection, then save as grayscale PNG
  2. GIMP: Use Fuzzy Select or Paths tool, export selection as grayscale
  3. Photopea (free online): Same workflow as Photoshop
  4. Remove.bg: For subject isolation, use the alpha channel

Creating a Mask Image

  1. Open your image in an editor
  2. Create a new layer filled with black
  3. Paint white on areas where you want HDR
  4. Export as PNG (grayscale or with alpha)
  5. Upload using "Upload" button

Tip: The mask will be automatically scaled to match your image dimensions.
